Ubuntu 18.04 Desktop 安装搜狗输入法

发表于 2018-12-05 分类于 software

Ubuntu 18.04 Desktop 自带 iBus 输入法，其中也支持中文拼音、五笔等输入法。

经过一天的使用，突然发现无法输入中文了，即使调整到中文状态也无法打出汉字。于是打算使用 Linux 下中文输入相对支持较好的——搜狗输入法 for Linux 。

安装 Fcitx

搜狗拼音依赖 Fcitx ，因此首先需安装 Fcitx 。（已安装可跳过）
使用以下命令安装 Fcitx：

sudo apt-get install -y fcitx-bin
sudo apt-get install -y fcitx-table

安装 搜狗输入法 for Linux

下载

访问 搜狗输入法For Linux 官网，点击下载 64bit 版（或 32bit 版）。

安装

安装包文件为 deb 包，可用 GDebi 安装。

//安装 GDebi
sudo apt-get install -y gdebi

方法1. 双击下载的安装包，点击安装，输入密码即可。

方法2. 或使用终端安装：

//安装包及其路径以 /home/Ubuntu/Downloads/sogoupinyin_2.2.0.0108_amd64.deb 为例
cd /home/Ubuntu/Downloads
sudo gdebi sogoupinyin_2.2.0.0108_amd64.deb
或
sudo dpkg -i sogoupinyin_2.2.0.0108_amd64.deb

配置 Fcitx

在 语言支持 中修改 键盘输入法系统 为 fcitx，重启系统。
重启后，找到 Fcitx 配置 ，添加搜狗拼音，并对输入法顺序作调整。

此时搜狗输入法应该已经可以使用了。
