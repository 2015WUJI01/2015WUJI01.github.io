软件版本符号说明

发表于 2019-12-12 分类于 常识

每次都不记得 ^、~ 的区别……

符号说明

版本号格式为 major.minor.patch，以下用 m.m.p 代替

  1. 直接指定

    • m.m.p

  2. 大小于 > >= < <=

    • >=m.m.p
    • <m.m.p
    • …

  3. 最右侧任意 ~

    • ~m：>=m.0.0，且 minor 及 patch 任意

      • ~1

    • ~m.m：>=m.m.0，且 patch 任意

      • ~1.0
      • ~1.2

    • ~m.m.p：>=m.m.p，且 patch 为大于 p 的任意值

      • ~1.0.0
      • ~1.2.0
      • ~1.0.2

  4. 非 0 右侧任意 ^，从左向右，第一个非 0 号的右侧任意

    • ^0.0.p：相当于直接指定 0.0.p
      • ~0.0.1
    • ^0.m.p：major 固定，minor 固定，patch 为大于 p 的任意值
      • ~0.2.3
    • ^m.m.p

  5. 任意符 x

    • m.m.x：x 位置表示任意

  6. 版本任意 *，留空

    • *
    • 留空

  7. 版本区间范围，包含首尾版本号

    • m.m.p-M.M.P：版本 >=m.m.p 且 <=M.M.P

  8. 或运算符 ||，version1||version2||...version 版本要求满足其一即可

    • 1.1.1-1.2.9 || >=3.5.0 || ^0.1.2
